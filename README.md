# Wallet API

Finaya Digital Wallet.

| Technology | Version |
|   :---:    |  :---:  |
| Java       | 21      |
| Spring     | 4.0.1   |
| PostgreSQL | 15      |

### How to run locally?

1. Open a terminal in the application's root directory and run: `docker compose up -d`
2. Open your browser and go to the address http://localhost:8080/
